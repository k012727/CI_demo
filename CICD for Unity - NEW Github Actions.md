建立git
------
1. 在github上建立專案

2. 在本地端新增.gitignore

3. 內容要google "gitignore.io"
https://www.toptal.com/developers/gitignore
輸入"Unity" 搜尋

4. 新增.gitignore 貼上code

5. 建立local端git 連結github

6. git
```
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin github連結
git push -u origin main
```
